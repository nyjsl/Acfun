# LewReorderableLayout
一个 UICollectionViewLayout，长按可以拖拽排序，同时支持纵向和横线滚动。

- ![效果图](https://github.com/pljhonglu/LewReorderableLayout/blob/master/demo.gif)

# 使用方法

请参考 Demo 中的代码

# 所需环境

- ARC
- iOS7 及以上
